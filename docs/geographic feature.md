# geographic feature

abstraction of real-world phenomena

NOTE: Geographic features can be defined for any geographic entity
            that can be defined with geospatial boundaries, such as landmarks, buildings, transport
            infrastructure (e.g., road, pathway, railway, bridge structures, intersections, traffic
            signal cabinet, traffic signal pole, waterway), jurisdictional areas (e.g., country,
            city), etc.
