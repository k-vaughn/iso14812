# sequential

operational model (3.5.9.1) where services (3.5.1.1) are provided to a
            single user (3.5.1.3) at any one time

NOTE: A single transport consumer can request the transport of
            multiple passengers (3.6.2.1).

EXAMPLE: Traditional taxi service.
