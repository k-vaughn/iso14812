# road

curvilinear length of roadway (3.3.1.3) that shares the same road
            identifier (3.3.5.13)

NOTE: A length of the road network (3.3.5.2) may be referenced by
            multiple designators (e.g. Main Street and Route 7). In this case, there would be two
            "roads" that share the same set of "carriageways" (3.3.1.5).
