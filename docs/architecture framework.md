# architecture framework

conventions, principles and practices for the description of architectures
            (3.1.3.1) established within a specific domain of application and/or community of
            stakeholders (3.1.3.4)

EXAMPLE: Generalised Enterprise Reference Architecture and Methodologies
            (GERAM) [ISO 15704] is an architecture framework.
