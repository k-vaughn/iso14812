# planning architecture

architecture (3.1.3.1) that provides a long-term vision of system elements
            (3.1.3.10) that may be deployed and managed by different projects and/or entities
            (3.1.1.1) within a geographic area

NOTE: Some countries use the term "regional architecture", but in
            International Standards, the term "regional" is avoided due to its multiple meanings.
