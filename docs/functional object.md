# functional object

set of related processes (3.1.7.2) that are performed by a physical object
            (3.1.8.1) to fulfil aspects of an ITS service (3.5.3.1)

NOTE: The term "module" is used by the European FRAME architecture
            while the Architecture Reference for Cooperative and Intelligent Transportation (ARC-IT)
            uses the term "functional object".

EXAMPLE: A vehicle OBE can include a "vehicle basic safety" functional object.
