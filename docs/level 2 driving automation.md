# level 2 driving automation

driver support system (3.7.3.7) that provides sustained lateral and
            longitudinal vehicle (3.7.1.1) motion control within a specific operational design
            domain (3.7.3.2) with the expectation that a conventional driver (3.6.2.4) completes the
            object and event detection and response

NOTE: Other driving automation levels include "level 1 driving
            automation" (3.7.3.8), "level 3 ADS" (3.7.3.11), "level 4 ADS" (3.7.3.12), and "level 5
            ADS" (3.7.3.13).
