# design speed

<vehicle> maximum speed (3.7.2.1) at which a vehicle (3.7.1.1) is
            intended to operate for a sustained period of time

NOTE: The term "top speed" is defined in SAE J3194 in relation to
            powered micromobility vehicles and is considered to be semantically equivalent to design
            speed.
