# vulnerable road user

road user (3.6.1.2) who is not protected by the body of a motor vehicle
            that meets local safety regulations

EXAMPLE: Pedestrian (3.6.1.4), cyclist, motorcyclist, road work personnel,
            horse, horseback rider, person with disabilities (e.g., wheelchair), etc.
