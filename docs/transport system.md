# transport system

system (3.1.2.1) of infrastructure elements (3.1.3.10) and optionally
            vehicles (3.7.1.1) that are jointly designed to move material entities (3.1.1.3) from an
            origin to a destination

NOTE: Transport systems can also include any supporting system, such
            as information and control systems.
