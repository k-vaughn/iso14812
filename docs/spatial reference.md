# spatial reference

description of a spatial location (3.4.1.2) in the real world according to
            a defined reference system

NOTE: It is not necessary for the rules be formal coordinates but they
            could be descriptive.

EXAMPLE: Coordinate tuple: 51.476852, -0.000500.
