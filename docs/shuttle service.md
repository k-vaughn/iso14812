# shuttle service

shared transport service (3.5.6.2) that transports passengers (3.6.2.1)
            between two specified locations (3.4.1.1)

NOTE: Each location can be defined as point, linear or area locations
            (3.4.1.6). However, the areas of the two locations should not overlap.
