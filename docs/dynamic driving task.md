# dynamic driving task

all real-time operational and tactical functions required to operate a
            vehicle (3.7.1.1) in on-road traffic

NOTE: The DDT includes lateral vehicle motion control, longitudinal
            vehicle motion control, monitoring the driving environment (3.1.3.11), object and event
            response execution, manoeuvre planning and enhancing conspicuity.
