# built-in vehicle input device

input device (3.7.1.4) designed to be physically connected (3.2.3.6) to a
            vehicle (3.7.1.1) and to remain connected even when the vehicle is not in use

NOTE: Built-in input devices include devices that can be temporarily
            disconnected (3.7.4.3) for security reasons (e.g. some radios are equipped with
            detachable front panels).
