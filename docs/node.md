# node

<road network> component of a road model (3.3.5.3) representing a
            point location (3.4.1.3) of the road network (3.3.5.2) graph

NOTE: The point location typically represents a point along the road
            (3.3.5.1) which can be used to designate the location (3.4.1.1) of a junction (3.3.6.2),
            public transport (3.5.6.1) stop, jurisdictional boundary, termination point, etc.
