# DDT performance-relevant system failure

malfunction in a vehicle system (3.2.1.5) that prevents the driving
            automation system (3.7.3.6) from reliably performing its portion of the DDT (3.7.3.1) on
            a sustained basis

NOTE: The malfunction can be internal to the driving automation system
            or part of another vehicle system.
