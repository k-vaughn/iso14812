# vehicle input device

material entity (3.1.1.3) that can affect the vehicle's (3.7.1.1) operation

NOTE: Input devices include those used to control the motion of the
            vehicle (e.g. a brake pedal), the state of vehicular equipment (e.g. headlight control),
            the configuration of the vehicle (e.g. temperature control), etc.

EXAMPLE: steering wheel, lever, pedal, knob, button, touch screen
