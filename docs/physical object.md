# physical object

<physical view> abstraction of a material entity (3.1.1.3) that
            interacts with other abstract material entities in the provision of ITS services
            (3.5.3.1)

NOTE: Physical objects are represented as elements (3.1.3.10) within
            the physical view (3.1.4.7) and perform a role. Physical objects can be implemented as
            cloud-based systems (3.1.2.1).
