# public transport app

mobility app (3.5.13.1) that assists a transport user (3.5.2.3) in using a
            public transport (3.5.6.1) system (3.1.2.1)

NOTE: A public transport app can include features such as viewing
            maps, searching routes and schedules, real-time arrival information, ticketing,
            electronic boarding pass, etc.
