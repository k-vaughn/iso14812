# terminator

entity (3.1.1.1) that is external to the ITS service (3.5.3.1)
            implementation but with which the implementation communicates either to obtain inputs or
            to which it can send outputs

NOTE: A terminator can exist within functional (3.1.4.5) and physical
            views (3.1.4.7).
