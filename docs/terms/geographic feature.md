Clause: 3.4.1.7

Note 1 to entry: Geographic features can be defined for any geographic entity that can be defined with geospatial boundaries, such as landmarks, buildings, transport infrastructure (e.g., road, pathway, railway, bridge structures, intersections, traffic signal cabinet, traffic signal pole, waterway), jurisdictional areas (e.g., country, city), etc.

Source: ISO 19101-1:2014, 4.1.11, "geographic" has been added to the term to clarify context; note 1 to entry has been replaced.

History note: Introduced in ISO/TS 14812:2025
