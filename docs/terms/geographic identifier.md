# geographic identifier

[spatial reference](spatial reference.md) in the form of a label or code that identifies a [location](location.md)

</object>

Clause: 3.4.2.6

Deprecated term: location code

EXAMPLE: "Spain" is an example of a label (country name); "SW1P 3AD" is an example of a code (postcode).

Note 1 to entry: The term "location code" has been used previously in ISO/TC 204 documents, but "geographic identifier" is preferred to better align with the activities of ISO/TC 211.

Source: ISO 19112:2019, 3.1.2, modified - Note 1 to entry and deprecated term added.

History note: Introduced in ISO/TS 14812:2022
