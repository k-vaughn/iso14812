# implementation

the process or the result of the process of translating a design into hardware components, software components, or both

<object type="image/svg+xml" data="../../diagrams/implementation.dot.svg">
    <img alt="implementation Diagram" src="../../diagrams/implementation.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.1.15.2

EXAMPLE: Example: The process of building a specific model of a smartphone or the resultant model of smartphone

Note 1 to entry: An implementation is the conceptual set of all instances of a product resulting from the translation. The set of all instances can consist of a single instance.

Source: ISO/IEC/IEEE 24765:2017, definitions 1 and 2 merged with example and note added

History note: Introduced in ISO/TS 14812:2025
