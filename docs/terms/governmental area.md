# governmental area

transport-related, [jurisdictional area](jurisdictional area.md) whose only [jurisdictional entities](jurisdictional entity.md) are governmental entities

</object>

Clause: 3.4.3.2

EXAMPLE: A subway station owned by a governmental entity is a governmental area.

Note 1 to entry: The boundaries of a jurisdictional area can include or exclude geofenced areas to indicate which jurisdictional entities have authority over [transport facilities](transport facility.md) as needed.

History note: Introduced in ISO/TS 14812:2025
