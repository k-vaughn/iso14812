# hard shoulder

part of the paved [carriageway](carriageway.md) designed to support traffic loads but not normally intended for driving

<object type="image/svg+xml" data="../../diagrams/hard shoulder.dot.svg">
    <img alt="hard shoulder Diagram" src="../../diagrams/hard shoulder.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.3.1.19

Note 1 to entry: A hard shoulder can be narrow or wide enough for a [traffic lane](traffic lane.md).

Note 2 to entry: A hard shoulder can have a surface that discourages usage as a driving surface in some countries.

History note: Introduced in ISO/TS 14812:2022
