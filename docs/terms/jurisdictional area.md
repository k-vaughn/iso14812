# jurisdictional area

territory for which a [jurisdictional entity](jurisdictional entity.md) has the power to define rules of behaviour

Clause: 3.4.3.5

Note 1 to entry: A jurisdictional area might include non-contiguous locations.

History note: Introduced in ISO/TS 14812:2025
