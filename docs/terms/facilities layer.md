# facilities layer

communications layer in the [ITS station reference architecture](ITS-S reference architecture.md) corresponding to the session, presentation and application layers of the OSI model

</object>

Clause: 3.1.5.5

Alternative preferred term: ITS-S facilities layer

Deprecated term: application layer

Note 1 to entry: Within the US, the NTCIP standards call the facilities layer the "application layer". However, as this term is easily confused with both the OSI application layer and the [application entity](application entity.md), the term should be avoided and qualified when used (e.g. OSI application layer).

History note: 2025: Revised based on input from WG 16 and to align more closely with the other definitions within ISO 14812.

History note: Introduced in ISO/TS 14812:2022.
