# shoulder

either a [hard shoulder](hard shoulder.md) or a [soft shoulder](soft shoulder.md)

Clause: 3.3.1.18

Note 1 to entry: In some jurisdictions, a shoulder can be used for emergency stops and/or other purposes in addition to providing a clear zone for safety purposes.

Note 2 to entry: [Roadside](roadside.md), shoulder and [verge](verge.md) define similar but slightly different concepts

History note: Introduced in ISO/TS 14812:2022

## Specializations of shoulder

| Class | Description |
| --- | --- |
| [hard shoulder](hard shoulder.md) | part of the paved [carriageway](carriageway.md) designed to support traffic loads but not normally intended for driving |
| [soft shoulder](soft shoulder.md) | unpaved part of the [road reservation](road reservation.md) not belonging to the [carriageway](carriageway.md) but providing lateral support to it |
