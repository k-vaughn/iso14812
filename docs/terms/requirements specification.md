# requirements specification

document that specifies the requirements for a system or component

Clause: 3.1.15.6

EXAMPLE: Example: Wi-Fi 7, 5G, Bluetooth 5.3

Note 1 to entry: A device can conform to multiple requirements specifications, each with a different focus.

Source: ISO/IEC/IEEE 24765:2017, example and note added

History note: Introduced in ISO/TS 14812:2025
