# contractual model

legal framework for the agreement between a [service provider](service provider.md) and a [user](user.md)

<object type="image/svg+xml" data="../../diagrams/contractual model.dot.svg">
    <img alt="contractual model Diagram" src="../../diagrams/contractual model.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.5.7.1

Alternative preferred term: shared mobility contract model

Note 1 to entry: Contracts can be via a third party, implied or verbal, especially in the case of [peer-to-peer](peer-to-peer.md).

History note: Introduced in ISO/TS 14812:2022
