# instance

discrete, bounded object with an intrinsic, immutable, and unique identity

Clause: 3.1.15.3

EXAMPLE: Example: A specific smartphone (e.g., having a unique serial number)

Note 1 to entry: An instance of an implementation can also be called a physical instantiation, in the case of a hardware component.

Source: ISO/IEC/IEEE 24765:2017, definition 1, example and note added

History note: Introduced in ISO/TS 14812:2025
