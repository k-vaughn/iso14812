particular place or position

Clause: 3.4.1.1

Note 1 to entry: The term "location" by itself is a broad term that can be used to reference spatial, computer memory, imaginary and other locations.

History note: Introduced in ISO/TS 14812:2022

| [point location](point location.md) | [spatial location](spatial location.md) with no length in any of the spatial dimensions |
| [spatial location](spatial location.md) | [location](location.md) within three-dimensional space |
