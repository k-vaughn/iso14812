# shared space

public area space for use by users with distinctly different characteristics

Clause: 3.3.3.8

EXAMPLE: facility designed for the use of pedestrians, cycles, and motor vehicles

Note 1 to entry: The space can be for travel, gatherings, rest, or other purposes.

History note: Introduced in ISO/TS 14812:2025
