# vehicle input device

[material entity](material entity.md) that can affect the [vehicle's](vehicle.md) operation

<object type="image/svg+xml" data="../../diagrams/vehicle input device.dot.svg">
    <img alt="vehicle input device Diagram" src="../../diagrams/vehicle input device.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.7.1.4

Alternative preferred term: input device

EXAMPLE: steering wheel, lever, pedal, knob, button, touch screen

Note 1 to entry: Input devices include those used to control the motion of the vehicle (e.g. a brake pedal), the state of vehicular equipment (e.g. headlight control), the configuration of the vehicle (e.g. temperature control), etc.

History note: Introduced in ISO/TS 14812:2022

## Specializations of vehicle input device

| Class | Description |

| [nomadic vehicle input device](nomadic vehicle input device.md) | [input device](input device.md) that can be carried into, or in near proximity of, a [vehicle](vehicle.md) and [connected](connected.md) as desired |
| [remote vehicle input device](remote vehicle input device.md) | [input device](input device.md) designed to be electronically [connected](connected.md) to a [vehicle](vehicle.md) even when the vehicle is not in close proximity |
