# single carriageway

having exactly one [carriageway](carriageway.md)

Clause: 3.3.1.7

Note 1 to entry: A single carriageway [road segment](road segment.md) could have multiple [lanes](lane.md) and could allow travel in opposite directions

Note 2 to entry: The "single carriageway" term can be used to describe a [road](road.md), [road section](road section.md), [road link](road link.md), or [road segment](road segment.md).

History note: 2025: Revised to change term to "single carriageway" rather than "single carriageway road segment" to broaden its use and revised definition and note 1 to entry accordingly. Removed admitted term "undivided highway" and associated note to entry.

History note: Introduced in ISO/TS 14812:2022
