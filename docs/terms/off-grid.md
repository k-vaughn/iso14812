# off-grid

characteristic of a [material entity](material entity.md) that is currently unable to send data to or receive data from [connected](connected.md) entities

</object>

Clause: 3.7.4.1

Alternative preferred term: ITS off-grid

History note: Introduced in ISO/TS 14812:2022

## Specializations of off-grid

| Class | Description |
| --- | --- |
| [disconnected](disconnected.md) | characteristic of a [material entity](material entity.md) that is or was equipped with communications equipment that is sufficiently physically disconnected such that the material entity is no longer able to send data to or receive data from [connected](connected.md) entities |
| [unconnected](unconnected.md) | characteristic of a [material entity](material entity.md) that is equipped with communications equipment that is not active or is otherwise unable to send data to or receive data from [connected](connected.md) entities |
| [unequipped](unequipped.md) | characteristic of a [material entity](material entity.md) that is not and has never been equipped with communications equipment for sending data to or receiving data from [connected](connected.md) entities |
