# data frame

specific grouping of [data elements](data element.md) that describes information of interest through a useful grouping of more atomic properties about one or more [classes](class.md)

<object type="image/svg+xml" data="../../diagrams/data frame.dot.svg">
    <img alt="data frame Diagram" src="../../diagrams/data frame.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.1.11.4

Note 1 to entry: A data frame can contain other data frames.

Note 2 to entry: The grouping can be a set, sequence or a choice.

History note: Introduced in ISO/TS 14812:2022
