# built-in vehicle input device

[input device](input device.md) designed to be physically [connected](connected.md) to a [vehicle](vehicle.md) and to remain connected even when the vehicle is not in use

Clause: 3.7.1.5

Alternative preferred term: built-in input device

Note 1 to entry: Built-in input devices are typically considered to be a part of the vehicle.

Note 2 to entry: Built-in input devices include devices that can be temporarily [disconnected](disconnected.md) for security reasons (e.g. some radios are equipped with detachable front panels).

Note 3 to entry: Built-in input devices require a maintenance operation to connect or disconnect.

History note: Introduced in ISO/TS 14812:2022
