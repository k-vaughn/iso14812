# coordinate tuple

ordered list of numbers that jointly designate the position of a point

Clause: 3.4.1.10

Note 1 to entry: The number of values in a coordinate tuple is immutable.

Note 2 to entry: This definition was derived by consolidating the definitions of "coordinate tuple", "coordinate", and "tuple" from ISO 19136:2020.

History note: Introduced in ISO/TS 14812:2025
