# spatial location

[location](location.md) within three-dimensional space

<object type="image/svg+xml" data="../../diagrams/spatial location.dot.svg">
    <img alt="spatial location Diagram" src="../../diagrams/spatial location.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.4.1.2

Alternative preferred term: ITS spatial location

Note 1 to entry: The "location" could be a point (consuming no space) or could consume space in one, two, or three dimensions. In any case, the location is identified within a larger space.

Note 2 to entry: Within ITS, locations are typically locations on or near the surface of the earth.

History note: Introduced in ISO/TS 14812:2022

## Specializations of spatial location

| Class | Description |
| --- | --- |
| [area destination](area destination.md) | area location, which is the destination of a defined route or itinerary |
| [area location](area location.md) | [spatial location](spatial location.md) enclosed within a two-dimensional boundary or boundaries across a defined surface |
| [geographic point location](geographic point location.md) | well defined geographic place described by one coordinate tuple |
| [linear location](linear location.md) | [spatial location](spatial location.md) that extends between two [point locations](point location.md) along a defined path |
| [point destination](point destination.md) | point location, which is the destination of a defined route or itinerary |
| [point location](point location.md) | [spatial location](spatial location.md) with no length in any of the spatial dimensions |
