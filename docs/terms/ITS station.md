# ITS station

[bounded secured managed domain](bounded secured managed domain.md) that is able to meet requirements of the [ITS trust domain](ITS trust domain.md) within which it is designed to participate

Note 1 to entry: The ITS station reference architecture is defined in ISO 21217.

Note 2 to entry: This complements the ISO 21217:2020 definition (which identifies the component parts) and is consistent with the text of ISO 21217:2020.

History note: 2025: Revised to indicate the "design" and to add Note 2 to entry.

History note: Introduced in ISO/TS 14812:2022.
