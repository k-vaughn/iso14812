# nomadic vehicle input device

[vehicle input device](vehicle input device.md) that can be carried into, or in near proximity of, a [vehicle](vehicle.md) and [connected](connected.md) as desired

    <img alt="nomadic vehicle input device Diagram" src="../../diagrams/nomadic vehicle input device.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.7.1.6

Alternative preferred term: nomadic input device

EXAMPLE: A smartphone connected to a vehicle (via USB or Bluetooth) to provide [driver](driver.md) navigation on the vehicle's large screen display

Note 1 to entry: NDs are often more closely associated with a [person](person.md) than they are with the vehicle.

Note 2 to entry: Nomadic input devices do not require a maintenance operation to connect or disconnect.

History note: Introduced in ISO/TS 14812:2022
