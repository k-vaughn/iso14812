# ITS implementation

integration of each [physical object](physical object.md) necessary to implement one or more [ITS applications](ITS application.md)

</object>

Clause: 3.2.8.5

Note 1 to entry: An ITS application typically requires multiple components (e.g. an ITS-S acting as a user and another ITS-S acting as a provider). An ITS implementation includes a sample of each component necessary for the service but often does not represent a complete deployment.

Note 2 to entry: An ITS implementation is typically used for a laboratory or other experimental [environment](environment.md) prior to a full-scale deployment.

History note: Introduced in ISO/TS 14812:2022
