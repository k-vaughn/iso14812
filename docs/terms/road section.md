# road section

aggregation of one or more [road links](road link.md) that represents a contiguous length of a [road](road.md) that shares the same management and operational strategies

</object>

Clause: 3.3.5.7

Example 1: A speed limit regulation can be applied to a road section.

Example 2: A traffic signal timing plan is applied to one road section.

Note 1 to entry: Different [road models](road model.md) can divide the same [road](road.md) into different road sections.

History note: 2025: Added example 2.

History note: Introduced in ISO/TS 14812:2022
