# dynamic driving task

all real-time operational and tactical functions required to operate a [vehicle](vehicle.md) in on-road traffic

</object>

Clause: 3.7.3.1

Alternative preferred term: DDT

Note 1 to entry: The DDT excludes strategic functions such as trip scheduling and selection of destinations and waypoints.

Note 2 to entry: The DDT includes lateral vehicle motion control, longitudinal vehicle motion control, monitoring the driving [environment](environment.md), object and event response execution, manoeuvre planning and enhancing conspicuity.

History note: Introduced in ISO/TS 14812:2022
