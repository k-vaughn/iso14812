# soft shoulder

unpaved part of the [road reservation](road reservation.md) not belonging to the [carriageway](carriageway.md) but providing lateral support to it

Note 1 to entry: Soft shoulders are not normally intended for driving.

Note 2 to entry: Soft shoulders often have a grass or gravel surface.
