# financial model

economic framework for the agreement between a [service provider](service provider.md) and a [user](user.md)

    <img alt="financial model Diagram" src="../../diagrams/financial model.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.5.8.1

History note: Introduced in ISO/TS 14812:2022

## Specializations of financial model

| Class | Description |
| --- | --- |
| [commercial](commercial.md) | type of [financial model](financial model.md) where [services](service.md) are provided with a profit motive |
| [cooperative](cooperative.md) | type of [financial model](financial model.md) where [users](user.md) partner to defray the costs of [services](service.md) |
| [fractional ownership](fractional ownership.md) | <transport service> [cooperative](cooperative.md) [financial model](financial model.md) where [users](user.md) partner to support the purchase, maintenance and overhead costs associated with a [service](service.md) |
| [mutual benefit](mutual benefit.md) | [cooperative](cooperative.md) [financial model](financial model.md) where [users](user.md) partner on a per-use basis |
| [private](private.md) | <transport service> [financial model](financial model.md) where [services](service.md) are provided to members of the owning organization without usage fees |
| [public](public.md) | type of [financial model](financial model.md) where [services](service.md) are provided by an administrative authority in an attempt to better serve societal interests |
