# sequential

[operational model](operational model.md) where [services](service.md) are provided to a single [user](user.md) at any one time

Clause: 3.5.9.2

EXAMPLE: Traditional taxi service.

Note 1 to entry: A single transport consumer can request the transport of multiple [passengers](passenger.md).
