# road identifier

designator by a road authority to uniquely identify one or more curvilinear lengths of [roadway](roadway.md) within the context of the road authority

<object type="image/svg+xml" data="../../diagrams/road identifier.dot.svg">
    <img alt="road identifier Diagram" src="../../diagrams/road identifier.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.3.5.12

Note 1 to entry: A single road identifier can be assigned to non-contiguous lengths of roadway.

Note 2 to entry: The identifier is typically either a name (e.g., Main Street), a class of road coupled with a number (e.g., "M25" meaning "motorway no. 25").

Note 3 to entry: The same road identifier can be allocated by different road authorities to different, physically distinct, curvilinear lengths of roadway.

History note: Introduced in ISO/TS 14812:2025
