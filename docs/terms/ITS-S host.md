# ITS-S host

[ITS-S node](ITS-S node.md) comprised of [ITS-S](ITS station.md) functionalities other than the functionalities of an [ITS-S router](ITS-S router.md), ITS-S border router, ITS-S mobile router or an ITS-S gateway

Clause: 3.2.7.6

Alternative preferred term: management host

Note 1 to entry: ITS-S border router and ITS-S mobile router are ITS-S routers with added functionalities, which is why they are separately excluded in the above definition.

Note 2 to entry: ITS-S border router, ITS-S mobile router, and ITS-S gateway are defined in ISO 21217.

History note: 2025: Revised to be an ITS-S node.

History note: Introduced in ISO/TS 14812:2022
