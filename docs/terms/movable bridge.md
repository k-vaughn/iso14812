# movable bridge

[bridge](bridge.md) with a deck that can be moved in a way that disrupts the continuous travel path

<object type="image/svg+xml" data="../../diagrams/movable bridge.dot.svg">
    <img alt="movable bridge Diagram" src="../../diagrams/movable bridge.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.3.7.4

Note 1 to entry: Movable bridges can be used over waterways to allow for the passage of tall ships.

Note 2 to entry: Movable bridges include, but are not limited to, bascule bridges, vertical lift bridges, and swing bridges.

History note: Introduced in ISO/TS 14812:2025
