# design speed

<vehicle> maximum [speed](speed.md) at which a [vehicle](vehicle.md) is intended to operate for a sustained period of time

<object type="image/svg+xml" data="../../diagrams/design speed.dot.svg">
    <img alt="design speed Diagram" src="../../diagrams/design speed.dot.png" /> <!-- Fallback for non-SVG browsers -->
</object>

Clause: 3.7.2.2

Alternative preferred term: vehicle design speed

Admitted term: top speed

Note 1 to entry: The term "top speed" is defined in SAE J3194 in relation to powered micromobility vehicles and is considered to be semantically equivalent to design speed.

History note: Introduced in ISO/TS 14812:2022
