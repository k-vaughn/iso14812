# ITS-S host

ITS-S node (3.2.7.8) comprised of ITS-S (3.2.7.3) functionalities other
            than the functionalities of an ITS-S router (3.2.7.7), ITS-S border router, ITS-S mobile
            router or an ITS-S gateway

NOTE: ITS-S border router and ITS-S mobile router are ITS-S routers
            with added functionalities, which is why they are separately excluded in the above
            definition.
