# governmental area

transport-related, jurisdictional area (3.4.3.4) whose only jurisdictional
            entities (3.4.3.6) are governmental entities

NOTE: The boundaries of a jurisdictional area can include or exclude
            geofenced areas to indicate which jurisdictional entities have authority over transport
            facilities (3.4.3) as needed.

EXAMPLE: A subway station owned by a governmental entity is a governmental
            area.
